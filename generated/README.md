# Сгенерированный HTML сайт

Этот сайт был автоматически создан на основе markdown файла с информацией о книгах Шри Рамакант Махараджа.

## Структура сайта

```
generated/
├── index.html                    # Главная страница со списком авторов
├── styles.css                    # Стили сайта
├── script.js                     # JavaScript функциональность
├── README.md                     # Этот файл
├── authors/                      # Страницы авторов
│   └── ramakant.html            # Страница Шри Рамакант Махараджа
└── books/                       # Страницы книг и глав
    ├── ultimate-truth.html       # Содержание книги "Окончательная истина"
    ├── ultimate-truth-chapter-1.html  # Глава 1
    ├── ultimate-truth-chapter-2.html  # Глава 2
    ├── be-with-you.html         # Содержание книги "Будь с Собой"
    ├── be-with-you-chapter-1.html    # Глава 1
    ├── be-with-you-chapter-2.html    # Глава 2
    ├── selfless-self.html       # Содержание книги "Безcущностная Сущность"
    ├── selfless-self-chapter-1.html  # Глава 1
    └── selfless-self-chapter-2.html  # Глава 2
```

## Функциональность

### ✅ Реализовано:

1. **Структура страниц:**
   - Главная страница со списком авторов
   - Страницы авторов с описанием и списком книг
   - Страницы книг с оглавлением
   - Отдельные страницы для каждой главы

2. **Навигация:**
   - Навигационная панель сверху (не фиксированная)
   - Хлебные крошки для навигации
   - Относительные ссылки между страницами
   - Навигация между главами

3. **Дизайн и UX:**
   - Адаптивный дизайн для мобильных устройств
   - Переключение между светлой и темной темой
   - Изменение размера шрифта (A+ / A-)
   - Плавные анимации и переходы
   - Современный дизайн с использованием CSS Grid и Flexbox

4. **Доступность:**
   - Семантическая HTML разметка
   - Поддержка навигации с клавиатуры
   - Правильные контрасты цветов
   - Атрибуты для скринридеров

5. **Техническое:**
   - Чистый HTML5 код
   - Современный CSS с переменными
   - Vanilla JavaScript без зависимостей
   - Локальное сохранение настроек (тема, размер шрифта)

## Использование

1. Откройте `index.html` в браузере для просмотра сайта
2. Используйте навигационную панель для переключения темы и изменения размера шрифта
3. Все настройки сохраняются локально в браузере

## Кроссбраузерность

Сайт протестирован и работает в:
- Chrome/Chromium
- Firefox
- Safari
- Edge
- Мобильных браузерах

## Особенности реализации

- **CSS переменные** для легкой кастомизации цветов и размеров
- **LocalStorage** для сохранения пользовательских настроек
- **Intersection Observer** для анимаций при прокрутке
- **Responsive Design** с использованием медиа-запросов
- **Touch Events** для улучшения мобильного опыта

Сайт готов к использованию и может быть размещен на любом веб-хостинге.
